Over mij

Front-end Developer

Wie ben ik? Ik ben Ruben. In mijn vrije tijd zit ik graag achter mijn laptop om creatief bezig te zijn met code. Die liefde voor design en techniek zet ik nu in om mooie en gebruiksvriendelijke websites te bouwen. Misschien binnenkort ook die van jou?
Waarom ben ik een front-end web developer geworden? Drie jaar geleden begon ik met het leren van de programmeertaal C#. Deze taal wordt vaak gebruikt voor game development, iets wat me enorm veel energie gaf: je ideeën razendsnel omzetten naar speelbare personages is magisch. Toch groeide ik langzaam uit die hobby. Game development kost ontzettend veel tijd, en dat maakte het lastig om projecten echt af te ronden. Web development bleek voor mij het perfecte alternatief. Ik kan nog steeds mijn creativiteit kwijt en ideeën visueel maken, maar in een veel korter tijdsbestek. Creativiteit uitdrukken via code blijft voor mij een groot plezier. Hopelijk kan ik die passie inzetten om jou te helpen aan een sterke, professionele website.
Wat maak ik
One-page pagina
Alle informatie op een pagina.

Een korte introductie van je bedrijf of jezelf, je diensten of tarieven, contactgegevens en een handig formulier. Alles op één overzichtelijke pagina waar bezoekers snel vinden wat ze zoeken.

Waarom een one-page pagina? Een website op één pagina werkt verrassend goed. Doordat alle informatie op een pagina staat is het voor de bezoeker van de website makkelijk om hier doorheen te scrollen. Een one-page website kan daarom ook uitstekend dienen als online visite kaartje, of een plek waar klanten snel in contact kunnen komen met je.
